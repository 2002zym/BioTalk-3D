# BioTalk-3D Dataset Terms of Use

By downloading or using BioTalk-3D, the user agrees to the following terms.

## Permitted Uses

The dataset may be used for academic research, education, and
non-commercial scientific evaluation, subject to applicable laws and
institutional ethical requirements.

## Restrictions

Users must not:

1. Attempt to identify or re-identify any participant.
2. Use the dataset for biometric identification or surveillance.
3. Use the dataset for medical diagnosis or clinical decision-making.
4. Redistribute the full dataset or provide access credentials to third parties.
5. Remove copyright, attribution, or ethical-use notices.
6. Use the dataset in a manner that harms participants or violates applicable law.
7. Represent the physiological signals as artifact-free or purely neural recordings.

## Citation

Publications using BioTalk-3D must cite the corresponding BioTalk-3D paper.

## No Warranty

The dataset is provided for research purposes without warranties of
accuracy, fitness for a particular purpose, or uninterrupted availability.

## Contact

Questions regarding permitted use should be submitted through the official
GitHub repository.
